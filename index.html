File Name Cleaner

ファイル名整形ツール

CSVを投入 → 漢字をローマ字に変換・記号除去・ハイフン→アンダースコア → ダウンロード

CSVファイルを投入（複数同時OK）
クリックしてファイル選択も可
元のファイル名 整形後のファイル名
整形ルール
  • 漢字・ひらがな・カタカナ → ヘボン式ローマ字に変換
  • ハイフン（-）→ アンダースコア（_）に変換
  • 英数字（A-Z a-z 0-9）とアンダースコア（_）以外の文字を除去
  • 連続するアンダースコアは1つに圧縮、前後のアンダースコアも除去
  • 拡張子（.csv）はそのまま維持
  • 整形後が空になった場合は「file_」＋ハッシュ4桁を使用
